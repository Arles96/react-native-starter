## What's inside

- Always up-to-date React Native and Expo scaffolding
- UI/UX Design from industry experts
- Modular and well-documented structure for application code
- Redux for state management
- React Navigation for simple navigation
- Disk-persisted application state caching
- More than 16 Ready-to-use Pages

## Getting Started

#### 1. Clone and Install

```bash
# Clone the repo
git clone https://github.com/flatlogic/react-native-starter.git

# Install dependencies
yarn install
```

#### 2. Open RNS with Expo

Install Expo XDE (https://expo.io/tools), open your cloned project there and press "Open on Device".

That's it! Cool, right?
